# Run Evaluation: General Quality

A catch-all sweep for things that feel "off", "broken", or "a bit weird" about this run — the stuff that doesn't fit cleanly under any of the dedicated dimensions. You're the residual.

## What to flag

Not exhaustive — use judgement. The usual suspects:

- **Structural oddness** — orphan pages, missing links that should exist, link types that don't match what the pages are doing, graph shapes that look confused.
- **Calibration vibes** — credence/robustness scores that feel obviously wrong relative to the evidence on the page. (The Calibration agent does per-claim audits; you're catching the cases a skim would notice.)
- **Headline discipline** — headlines that overreach the body, need the question next to them to make sense, or use vague framing where something concrete was possible.
- **Epistemic hygiene** — performative hedging, restating the question as if it were analysis, generating fluent text about something the run doesn't actually have a grip on, and the other failure modes called out in the preamble.

## How to work

Read through `[ADDED BY THIS RUN]` content; flag what strikes you. Cite page IDs for everything; a vague complaint without an anchor is not useful.

## Output

- **Summary** — 2–3 sentences.
- **Strengths** — things the run did well outside the excluded dimensions.
- **Issues** — each with page IDs and a one-line description of what's wrong.
- **Patterns** — recurring good or bad habits, only if you see one.
- **Overall** — one paragraph.
