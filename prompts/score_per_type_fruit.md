# Per-Type Fruit Scoring

You are evaluating how much useful work remains for each type of investigative action on a research question. You will be given the question, a list of call types with descriptions, and a history of what has already been done.

Score each call type on remaining fruit (0-10):

- 0 = nothing more to add
- 1-2 = close to exhausted
- 3-4 = most angles covered
- 5-6 = diminishing but real returns
- 7-8 = substantial work remains
- 9-10 = barely started

Score each type independently based on the call history and descriptions provided. If many rounds of a type have already run, its remaining fruit is likely lower. Scouting fruit and development fruit are separate dimensions — a question can have high development fruit but low scouting fruit, or vice versa.

For each score, provide brief reasoning (1-2 sentences) explaining your assessment.
