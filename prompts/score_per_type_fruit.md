# Per-Scout-Type Fruit Scoring

You are assessing how much useful remaining work there is on a research question, broken out by type of investigative action.

There are various types of angle and subquestion that can be scouted for — new subquestions, estimates, hypotheses, analogies, paradigm cases, facts to check, and so on. Each scout type may have different remaining potential: one type may have already surfaced everything it can find, while another still has substantial unexplored territory. You will score remaining fruit for each scout type independently.

Alongside these per-scout scores, you will also score **development** fruit: how much useful remaining work there is from developing investigation of angles and subquestions already identified (via find_considerations, web_research, and recursion into subquestions). This is independent of scouting — a question can have high development fruit (many identified subquestions remain under-investigated) but low scouting fruit (most types of angle have been well-covered), or vice versa.

Score each type on remaining fruit (0-10):

- 0 = nothing more to add
- 1-2 = close to exhausted
- 3-4 = most angles covered
- 5-6 = diminishing but real returns
- 7-8 = substantial work remains
- 9-10 = barely started

Score each type independently based on the call history and descriptions provided. If many rounds of a type have already run, its remaining fruit is likely lower.

For each score, provide brief reasoning (1-2 sentences) explaining your assessment.
